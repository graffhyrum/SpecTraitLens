Search every profession specialization and perk in one flat, filterable index — no more digging through spec trees.

## What it does

**Profession Trait Search** builds a searchable **Specialization index** from your profession spec trees: every **Specialization**, **Sub-specialization**, and **Trait/Perk** in one scrollable list with descriptions visible at once. Type a keyword, narrow with filters, then click any row to jump straight to that node in the Professions **Specializations** page.

Works with all retail professions that use the Dragonflight+ specialization system.

## Features

- **Full-tree index** — specializations, sub-specializations, and perks in one hierarchical list
- **Text search** — find perks and paths by name or description (e.g. `Multicraft`, `Resourcefulness`)
- **Major perks toggle** — when enabled, hide minor dial milestones and focus on capstone perks; turn off to show everything
- **Unearned toggle** — when enabled, show perks and paths you have not completed yet; turn off to show everything
- **Collapse / expand** — fold branches to skim a tree, or collapse all at once
- **Progress at a glance** — color-coded rows for earned, accessible, and locked content; rank badges on sub-specializations
- **Knowledge available** — unspent specialization currency for the selected profession
- **Click to navigate** — left-click a row to open Professions and select that specialization or sub-specialization

## How to open

| Method | Action |
|--------|--------|
| **Slash command** | `/pts` — toggle the standalone window |
| **Search from chat** | `/pts search Multicraft` — open with a search term pre-filled |
| **Minimap button** | Left-click — open index; right-click — settings |
| **Professions frame** | Side tab on **Recipes**, **Specializations**, or **Crafting Orders** — toggles an embedded index panel beside the frame |

The standalone window includes a profession dropdown when you have more than one specialization-enabled profession.

## Slash commands

- `/pts` — toggle the specialization index
- `/pts search <term>` — open with search text
- `/pts status` — current profession and visible row count
- `/pts help` — command list

## Why use it

Blizzard’s spec UI is built for spending points on one tree at a time. **Profession Trait Search** is built for **finding** things: which tree has Multicraft, what you still need to earn, and where a perk lives — without clicking through every branch manually.

## Requirements

- **WoW retail** — Dragonflight (10.0) or later
- No other addons required
